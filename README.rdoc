== README

This repository includes benchmarks for mass inserting data into database in Rails. The code was taken from:

https://www.coffeepowered.net/2009/01/23/mass-inserting-data-in-rails-without-killing-your-performance/


This rails project contains a postgresql, in order to compare the results with the above link, which were done with mysql db.

The main goal is to compare the results with pg_data_encoder, by Pete Brumm (https://github.com/pbrumm/pg_data_encoder), performance
